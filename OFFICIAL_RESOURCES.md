# Official Red Hat Resources (EX200)

## Exam Overview
The Red Hat Certified System Administrator (RHCSA) exam (EX200) tests your knowledge in areas of system administration common across a wide range of environments and deployment scenarios. The skills tested in this exam are the foundation for system administration across all Red Hat products.

- **Exam Code:** EX200
- **Time Limit:** 3 hours
- **Type:** Performance-based (you must perform tasks on a live system)
- **Passing Score:** 210 out of 300

## Core Objectives Categories (2024-2025)
1.  **Understand and use essential tools**: Access shell, output redirection, grep, SSH, file management.
2.  **Operate running systems**: Boot/reboot, process management, logs, systemd.
3.  **Configure local storage**: Partitions, LVM, file systems (vfat, ext4, xfs).
4.  **Create and configure file systems**: Permissions, ACLs, NFS.
5.  **Deploy, configure, and maintain systems**: Cron, software management (dnf, rpm), bootloader.
6.  **Manage basic networking**: IPv4/IPv6, hostname resolution.
7.  **Manage users and groups**: Creation, modification, password aging.
8.  **Manage security**: Firewall (firewalld), SELinux modes and contexts.
9.  **Manage containers**: Podman, container images, persistent storage.
