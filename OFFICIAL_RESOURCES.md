# Official Red Hat Resources

## Red Hat Certified System Administrator (RHCSA) - EX200

### Exam Overview
The **RHCSA (EX200)** tests your knowledge in areas of system administration common across a wide range of environments and deployment scenarios. The skills tested in this exam are the foundation for system administration across all Red Hat products.

- **Exam Code:** EX200
- **Time Limit:** 3 hours
- **Type:** Performance-based (live system tasks)
- **Passing Score:** 210 out of 300

### Key Links
*   **[Official Exam Objectives (v9/v10)](https://www.redhat.com/en/services/training/ex200-red-hat-certified-system-administrator-rhcsa-exam?section=Objectives)** - _Always check this for the latest requirements._
*   **[Red Hat Product Documentation](https://access.redhat.com/documentation/en-us/red_hat_enterprise_linux/)** - The definitive source for configuration details.
*   **[Red Hat Developer Program](https://developers.redhat.com/)** - Free RHEL Developer Subscription for personal use (great for labs).

### Core Objectives Categories
1.  **Understand and use essential tools**: Access shell, output redirection, grep, SSH, file management.
2.  **Operate running systems**: Boot/reboot, process management, logs, systemd.
3.  **Configure local storage**: Partitions, LVM, file systems (vfat, ext4, xfs).
4.  **Create and configure file systems**: Permissions, ACLs, NFS.
5.  **Deploy, configure, and maintain systems**: Cron, software management (dnf, rpm), bootloader.
6.  **Manage basic networking**: IPv4/IPv6, hostname resolution.
7.  **Manage users and groups**: Creation, modification, password aging.
8.  **Manage security**: Firewall (firewalld), SELinux modes and contexts.
9.  **Manage containers**: Podman, container images, persistent storage.

## Red Hat Certified Engineer (RHCE) - EX294

### Exam Overview
The **RHCE (EX294)** focuses on automation using Ansible. You must be an RHCSA to earn the RHCE certification. The exam tests your ability to use Ansible to automate the configuration of systems and applications.

- **Exam Code:** EX294
- **Time Limit:** 4 hours
- **Type:** Performance-based (live system tasks using Ansible)
- **Passing Score:** 210 out of 300

### Key Links
*   **[Official Exam Objectives (Reference)](https://www.redhat.com/en/services/training/ex294-red-hat-certified-engineer-rhce-exam?section=Objectives)**
*   **[Red Hat Enterprise Linux Automation with Ansible (RH294)](https://www.redhat.com/en/services/training/rh294-red-hat-enterprise-linux-automation-ansible)** - Official course.
*   **[Ansible Documentation](https://docs.ansible.com/)** - Essential reference for modules and core concepts.

### Core Objectives Categories
1.  **Understand core components of Ansible**: Inventories, modules, variables, facts, plays, playbooks, files, roles, collections.
2.  **Install and configure an Ansible control node**: Python3, configuration files, managed hosts/inventory.
3.  **Configure Ansible managed nodes**: SSH keys, privilege escalation.
4.  **Create simple shell scripts**: Using ad-hoc commands.
5.  **Create Ansible plays and playbooks**: Syntax (YAML), multiple plays, error handling.
6.  **Use modules for system administration** tasks: User management, cron, storage, network, services.
7.  **Work with roles and Ansible Content Collections**: System roles, creating roles.
