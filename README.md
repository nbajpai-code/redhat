# Red Hat Certification Exams (RHCSA & RHCE) Resources

Welcome to the comprehensive resource collection for Red Hat certification exams. This repository currently focuses on the core system administration and automation tracks.

## Supported Exams

### 1. [Red Hat Certified System Administrator (RHCSA) - EX200](OFFICIAL_RESOURCES.md#red-hat-certified-system-administrator-rhcsa---ex200)
The foundation for all Red Hat certifications. Covers essential system administration tasks.

### 2. [Red Hat Certified Engineer (RHCE) - EX294](OFFICIAL_RESOURCES.md#red-hat-certified-engineer-rhce---ex294)
Focuses on automation using Ansible. Prerequisite: RHCSA.

## Quick Start Guide

1.  **Review the Official Objectives**: Start with `OFFICIAL_RESOURCES.md` to know exactly what is tested for your target exam.
2.  **Choose a Primary Course**: Select a video course from `COURSES.md`.
3.  **Set Up Your Lab**:
    *   For **EX200**: Install RHEL 9 (free via Developer Subscription) in a VM.
    *   For **EX294**: Set up a control node and 2-3 managed nodes to practice Ansible.
4.  **Practice, Practice, Practice**: Use the `GITHUB_REPOS.md` for practice scenarios.
5.  **Get Unstuck**: Use `YOUTUBE_CHANNELS.md` for specific topic walkthroughs and `COMMUNITY_BLOGS.md` for troubleshooting.
